# RT Library Privacy Policy

Effective date: July 9, 2026

RT Library is a co-op shareholder-built catalog app for the shared library room at Riviera Towers West New York, a private co-op building.

## Data Collection

RT Library does not collect personal information in version 1.0. The app does not require an account, sign-in, location access, contacts, camera access, photo library access, microphone access, or notification permission.

## Catalog Data

The book catalog is bundled with the app and was generated from recent shelf photos using OCR and metadata lookup. Searches happen on the device.

## Network Content

When available, the app may load book cover images or metadata links from public catalog sources. RT Library does not use this to track users.

## Requests and Availability

Book requests and shared availability updates are marked Coming soon and are not submitted or synced in version 1.0.
