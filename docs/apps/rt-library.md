# RT Library

RT Library is a simple resident-built catalog for the shared library room at Riviera Towers West New York, a private residential building.

The app helps residents search and browse the building library by title, author, and genre without physically scanning every shelf.

## Current Release

- iOS app for iPhone and iPad
- Search by title, author, or genre
- Browse books by genre
- View book details, OCR confidence, source photo, and public metadata links when available
- Catalog generated from recent library shelf photos using OCR and metadata lookup
- 1,555 OCR-derived catalog entries across 30 genres in version 1.0

## Coming Soon

- Resident book request submissions
- Shared availability updates
- Book donation photo flow
- Improved metadata matching and catalog cleanup

Requests and availability updates are intentionally marked Coming soon in version 1.0 because shared resident storage is planned for a later release.

## Notes

The catalog is generated from photos and OCR, not manually entered one book at a time. Some entries may contain OCR mistakes or need resident review.
