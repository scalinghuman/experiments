# RT Library

RT Library is a simple co-op shareholder-built catalog for the shared library room at Riviera Towers West New York, a private co-op building.

The app helps co-op shareholders search and browse the building library by title, author, and genre without physically scanning every shelf.

## Current Release

- iOS app for iPhone and iPad
- Search by title, author, or genre
- Browse books by genre
- View book details, OCR confidence, source photo, and public metadata links when available
- Catalog generated from recent library shelf photos using OCR and metadata lookup
- 653 OCR-derived book candidates across 50 genres in version 1.0

## Coming Soon

- Co-op shareholder book request submissions
- Shared availability updates
- Book donation photo flow
- Improved metadata matching and catalog cleanup

Requests and availability updates are intentionally marked Coming soon in version 1.0 because shared storage for co-op shareholders is planned for a later release.

## Notes

The catalog is generated from photos and OCR, not manually entered one book at a time. The total includes verified and unverified book candidates, so some titles may contain OCR mistakes or need co-op shareholder review.
